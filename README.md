Projeto desenvolvido durante a NLW da Rocketseat. A pagina reune jogos favoritos, canais acompanhados e links sociais em uma interface visual com cards, animacoes e gradientes.

## Demo

[Acessar projeto online](https://rocketseat-nlw.vercel.app)

## Destaques

- Layout responsivo com HTML e CSS puro.
- Avatar do GitHub carregado dinamicamente.
- Secoes para jogos, canais e redes sociais.
- Animacoes de entrada e hover.
- Uso de imagens externas e SVGs para links sociais.

## Stack

| Tecnologia | Uso |
| --- | --- |
| HTML | Marcacao da pagina |
| CSS | Layout, animacoes e responsividade |
| Google Fonts | Tipografia |
| Twitch/YouTube assets | Imagens de jogos e canais |

## Como executar

Abra o arquivo `index.html` no navegador.

Para uma experiencia melhor durante desenvolvimento, use uma extensao como Live Server.

|-- styles.css

    `-- bg.jpg
